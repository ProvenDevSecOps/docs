<Note>You cannot use public emails like @gmail.com, @outlook.com, etc. Proven is focused on work and school accounts.</Note>

You can use Microsoft's [Sign in with Microsoft](https://account.microsoft.com/account) product to create a user account with Proven. 

- [Global](https://console.proven.cloud/signin)
- [US](https://console.us.proven.cloud/signin)
- [EU (Coming Soon)](https://console.eu.proven.cloud/signin)

After sign in/up, Azure Admins can see the `Proven Cloud` app inside the Azure [App Registrations page](https://portal.azure.com/#view/Microsoft_AAD_RegisteredApps/ApplicationsListBlade).

<Warning>If you get an `Invalid token/Id issuer error when using Sign In With Microsoft - please contact [Proven Support](/support) for resolution.</Warning>
